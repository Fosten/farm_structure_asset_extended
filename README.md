# farmOS Structure Asset Extended

This module is an add-on for the [farmOS](http://drupal.org/project/farm)
distribution.

## Features

Provides additional farm_structure_type_field_allowed_values to Structure Assets:
- concrete
- fire
- fuel
- metal
- treestand
- wood
